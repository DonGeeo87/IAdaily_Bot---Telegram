## 🎯 Promoción Automática

### Ejecutar Kit de Promoción

```bash
cd ~/telegram-ia-bot
python auto_promotion.py
```

Esto genera:
- ✅ Enlaces de compartir para todas las plataformas
- ✅ Mensajes promocionales pre-escritos
- ✅ Hashtags virales
- ✅ Estrategia de cross-promoción

### Archivos Generados

| Archivo | Función |
|---------|---------|
| `promotion_kit.json` | Kit completo de promoción |
| `social_posts/` | Posts para cada red social |

## 📱 Redes Sociales

### Publicador Automático

```bash
python social_poster.py
```

Genera posts para:
- **Twitter/X** - 280 chars con hashtags
- **LinkedIn** - Formato profesional
- **Instagram** - Caption con hashtags
- **TikTok** - Guion para videos
- **Facebook** - Post compartido
- **Reddit** - Formato para subreddits

| Instagram | 10:00, 15:00, 20:00 |
| TikTok | 11:00, 16:00, 21:00 |

## 🦠 Estrategias Virales

### 1. **Contenido Shareable**

```
✅ Tips cortos y accionables
✅ Herramientas gratuitas
✅ Comparativas "antes/después"
✅ Listas (Top 5, Top 10)
✅ Prompts listos para copiar
```

### 2. **Cross-Promotion**

**Subreddits para compartir:**
- r/InteligenciaArtificial
- r/IA
- r/ChatGPT
- r/MachineLearning
- r/artificial

**Grupos de Facebook:**
- "Inteligencia Artificial en Español"
- "IA y Machine Learning"
- "ChatGPT & AI Tools"

### 3. **TikTok Strategy**

```
📹 Formato: 30-60 segundos
🎬 Contenido: Demo de herramientas
📞 CTA: "Link en bio para más"
📅 Frecuencia: 1-2 videos/día
```

### 4. **Colaboraciones**

- Busca canales similares (no competidores directos)
- Propón intercambio de menciones
- Crea contenido conjunto
